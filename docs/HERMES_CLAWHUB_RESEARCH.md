## Install Hermes Agent
```bash
# Linux / macOS / WSL2 / Android (Termux)
curl -fsSL https://hermes-agent.nousresearch.com/install.sh | bash

# Windows (native)
iex (irm https://hermes-agent.nousresearch.com/install.ps1)
```

Then: `hermes setup --portal`

## Key Features to Learn
1. Skills System — Creates skills from experience, reuses across sessions
2. Memory System — Persistent memory that grows across sessions
3. SOUL.md — Define agent personality/voice globally
4. Context Files — Project-level context shaping every conversation
5. MCP Integration — Connect MCP servers to extend capabilities
6. Voice Mode — Real-time voice in CLI, Telegram, Discord
7. Tools & Toolsets — 60+ built-in tools (web search, image gen, TTS, browser)
8. Security — Command approval, authorization, container isolation
